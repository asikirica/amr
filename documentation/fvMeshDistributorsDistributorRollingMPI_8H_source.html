27Description
28 Dynamic mesh redistribution using the distributor specified in
29 decomposeParDict
30
31Usage
32 Example of single field based refinement in all cells:
33 \verbatim
34 distributor
35 {
36 type distributorRollingMPI;
37
38 libs ("libfvMeshDistributorsRollingMPI.so");
39
40 // How often to redistribute
41 redistributionInterval 10;
42
43 // Maximum fractional imbalance between the MPI rank loads
44 // before rebalancing

46

48 // before rebalancing
49 maxMemImbalance 0.1;
50
